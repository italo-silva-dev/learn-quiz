# Learn Quiz

Projeto desenvolvido com o objetivo de praticar programação e evoluir habilidades em desenvolvimento web.

## Sobre o projeto

O Learn Quiz começou como um projeto simples em HTML feito para aprender o básico, por exemplo os inputs, e evoluiu ao longo do tempo.
A ideia é criar uma plataforma onde usuários possam aprender, praticar e testar seus conhecimentos através de desafios (quiz).


### Evolução do projeto

• V1

- Projeto inicial em HTML
- Página única
- Estrutura simples
- Primeiro contato com desenvolvimento web
- Desenvolvida em Abril de 2024

• V2

- Separação em múltiplas páginas
- Uso de CSS para estilização
- Organização em pastas
- Tentativa de criar um "hub de desafios"
- Desenvolvida em Junho de 2024

• V3 (em desenvolvimento)

- Melhor uso de CSS (layout mais moderno)
- Implementação de JavaScript
- Sistema interativo de quizzes
- Possível integração com backend no futuro


#### Tecnologias utilizadas

- HTML

- (futuramente) JavaScript
- (futuramente) Backend
